# 📌 功能介绍
本脚本专为 天津市专业技术人员继续教育网公需课考试 开发，在考试页面一键提取所有题号、题干、单选/多选选项，自动整理格式并复制到剪贴板。
仅做题目内容展示与导出，不干预考试流程、不自动答题、不篡改页面数据，合规安全，可放心使用。
## 核心功能：
- 自动识别页面所有考试题目
- 精准抓取题号、题干、全部选项内容
- 统一排版整理，格式清晰，可直接粘贴记事本/文档
- 一键复制，无需手动摘抄，大幅提升备考效率

# 🔧 安装教程
## 前置要求
浏览器提前安装好 Tampermonkey（篡改猴） 插件

## 安装方法：
方法1：在 Tampermonkey 中创建一个新脚本，将代码粘贴到编辑器中并保存。

方法2：在 Tampermonkey 中搜索该脚本。

# 📖 使用方法
1. 打开天津专技继续教育公需课考试页面：https://gp.chinahrt.com/*
2. 页面右上角会自动显示 提取全部题目复制 悬浮按钮
3. 点击按钮，脚本自动扫描并提取页面所有题目
4. 提取完成自动弹窗提示，所有内容已复制到剪贴板
5. 直接粘贴到记事本、Word、文档中
